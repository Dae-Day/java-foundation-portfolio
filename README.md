# Student Grade Calculator

## Overview
A simple Java console app that calculates a student's average grade based on input marks.

## Tech Used
- Java 21
- Maven
- IntelliJ IDEA
- OOP Principles

## Features
- Input validation
- Average calculation
- Grade determination (A-F)
- Console report

## Learning Outcomes
- Java fundamentals & OOP
- Loops & conditionals
- Modular coding
